# CompteurDeMots 

## Description
Programme Java qui compte le nombre de mots 
dans un fichier texte.

## Fonctionnalités
- Lit un fichier texte ligne par ligne
- Affiche chaque ligne avec son numéro
- Compte les mots de chaque ligne
- Affiche le total de mots du fichier

## Comment exécuter
1. Ouvrir le projet dans IntelliJ IDEA
2. Exécuter la classe Main.java
3. Le programme affiche le nombre de mots
